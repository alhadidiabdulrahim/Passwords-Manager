# Passwords Manager

A simple **command-line Password Manager** written in Python that allows users to securely store, view, and manage passwords using hashed storage.

---

## Features

- **User Login**: Authenticate using a username and password.  
- **Add Passwords**: Save passwords for various sites securely.  
- **View Passwords**: Display all stored passwords in hashed format.  
- **Delete Passwords**: Remove stored passwords.  
- **Logout**: Exit the application safely.  

---

## Files

- **app.py**: Main application script that handles user interaction and program flow.  
- **Functions.py**: Contains all helper functions for login, password hashing, storage, and management.  
- **volt.json**: JSON file used to store user credentials and passwords (created automatically when the first password is added).
